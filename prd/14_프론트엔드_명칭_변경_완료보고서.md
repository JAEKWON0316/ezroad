# 프론트엔드 프로젝트명 변경 완료 보고서 (EzenRoad -> Linkisy)

## 개요
사용자의 요청에 따라 프론트엔드 애플리케이션의 프로젝트명을 "EzenRoad"에서 "Linkisy"로 성공적으로 변경하였습니다. 백엔드 코드는 변경하지 않았으며, 프론트엔드 UI 텍스트와 메타데이터만 수정되었습니다.

## 변경 사항 요약

### 1. UI 텍스트 변경
- **헤더 & 푸터**: 로고 텍스트를 `Linkisy`로 변경
- **메인 페이지**: 히어로 섹션, 서비스 소개, CTA 섹션의 텍스트 변경
- **로그인/회원가입 페이지**: 환영 메시지 및 폼 텍스트 변경
- **기타 페이지**: 공지사항, 약관, 개인정보처리방침, 사업자 약관, 문의하기 페이지 내 텍스트 변경
- **이메일 주소**: `support@ezenroad.com` -> `support@linkisy.com`

### 2. 메타데이터 및 구조 개선
- **Layout Refactoring**: `layout.tsx`를 서버 컴포넌트로 리팩토링하여 Next.js Metadata API를 지원하도록 개선
- **Metadata 적용**: HTML Title을 `Linkisy - 맛집 탐색 및 예약 플랫폼`으로 설정
- **Providers 분리**: 클라이언트 사이드 컨텍스트(`AuthProvider`, `Toaster`)를 `providers.tsx`로 분리
