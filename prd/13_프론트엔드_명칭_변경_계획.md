# 프론트엔드 프로젝트명 변경 계획 (EzenRoad -> Linkisy)

## 목표
프론트엔드 애플리케이션에서 사용자에게 노출되는 "EzenRoad" (및 관련 변형) 텍스트를 "Linkisy"로 변경합니다. 백엔드 코드는 변경하지 않습니다.

## 변경 대상

### 1. UI 텍스트 (User Interface Text)
- **헤더/푸터**: 로고 텍스트, 카피라이트 문구
- **메인 페이지**: 히어로 섹션 타이틀, 소개 문구
- **로그인/회원가입**: 환영 메시지, 폼 타이틀
- **기타 페이지**: 공지사항, 약관, 마이페이지 등

### 2. 메타데이터 (Metadata)
- **HTML Title**: 브라우저 탭에 표시되는 제목
- **Meta Description**: 검색 엔진 최적화(SEO) 설명 문구
- **Open Graph**: SNS 공유 시 표시되는 제목 및 설명

## 실행 전략
1. **백엔드 복구 확인**: 이전 단계에서 변경된 백엔드 폴더 구조가 원상복구 되었는지 확인합니다.
2. **텍스트 검색 및 치환**: `frontend` 디렉토리 내에서 `EzenRoad`, `이젠로드`, `EzRoad` 키워드를 검색하여 `Linkisy`, `링키지` 등으로 변경합니다.
    - 대소문자 구분하여 일괄 변경보다는 문맥을 확인하며 변경합니다.
    - 코드 변수명(예: `EzRoadComponent`)은 변경하지 않고, **표시되는 텍스트(String Literal)** 위주로 변경합니다.
3. **검증**: 브라우저 제어 도구를 사용하여 변경된 화면을 확인합니다.

## 주요 수정 예정 파일
- `frontend/src/app/layout.tsx` (메타데이터)
- `frontend/src/components/layout/Header.tsx`
- `frontend/src/components/layout/Footer.tsx`
- `frontend/src/app/page.tsx` (메인 페이지)
- `frontend/src/app/(auth)/**` (인증 관련 페이지)
