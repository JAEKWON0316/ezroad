## ⚠️ 주의사항

1. **테스트 계정은 공용입니다** - 다른 사용자와 함께 사용합니다.
2. **데이터가 변경될 수 있습니다** - 다른 사용자가 데이터를 수정할 수 있습니다.
3. **민감한 정보를 입력하지 마세요** - 개인정보나 민감한 데이터를 저장하지 마세요.
4. **주기적으로 초기화될 수 있습니다** - 테스트 데이터는 정기적으로 리셋될 수 있습니다.

## 🎯 테스트 시나리오 가이드

### 시나리오 1: 일반 사용자 플로우
1. `demo@ezroad.com`으로 로그인
2. 식당 목록에서 "테스트 맛집" 검색
3. 식당 상세 페이지에서 찜하기 👀
4. 예약 또는 웨이팅 신청
5. 마이페이지에서 예약/대기 현황 확인

### 시나리오 2: 사업자 플로우
1. `business@ezroad.com`으로 로그인
2. 파트너 대시보드에서 통계 확인
3. 예약 요청이 있으면 확정 처리
4. 웨이팅 손님 호출 및 착석 처리
5. 메뉴 가격이나 정보 수정해보기

### 시나리오 3: 리뷰 작성 플로우
1. `demo@ezroad.com`으로 로그인
2. 마이페이지 → 예약 내역
3. "완료됨" 상태의 예약에서 "리뷰 쓰기" 클릭
4. 별점과 후기 작성
